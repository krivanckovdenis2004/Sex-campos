Добавлен тестовый Import by URL.

Как работает:
1. Открой admin.html.
2. Вставь прямую ссылку на MP4.
3. Вставь ссылку на превью, если есть.
4. Нажми "Проверить ссылку".
5. Нажми "Добавить по ссылке".

Важно:
- Это работает с прямыми MP4-ссылками.
- Если сайт запрещает hotlink/CORS, видео может не проигрываться.
- Для постоянного проекта лучше скачивать/загружать легальный контент в свой storage/CDN.
